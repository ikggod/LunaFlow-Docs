달의 기록 개인정보 처리방침

최종 업데이트: 2025년 2월 11일 | 시행일: 2025년 2월 11일
중요: 달의 기록은 생리 주기 추적 앱으로, 매우 민감한 건강 정보를 다룹니다. 모든 데이터는 사용자의 기기에 로컬로 저장되며, 사용자의 명시적 동의 없이 외부로 전송되지 않습니다.

1. 개요

달의 기록(LunaFlow, "앱", "우리")은 DyCodeWorks에서 제공하는 생리 주기 추적 앱입니다. 사용자의 개인정보 보호를 최우선으로 생각하며, 본 개인정보처리방침은 달의 기록 앱이 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지 설명합니다.

2. 수집하는 정보

2.1 사용자가 직접 제공하는 정보

  • 생리 주기 정보: 생리 시작일, 종료일, 주기 길이
  • 증상 기록: 생리통, 두통, 복부 팽만 등
  • 기분 기록: 행복함, 슬픔, 짜증 등
  • 관계 기록: 성관계 날짜 및 피임 여부
  • 메모: 사용자가 입력한 개인 메모

2.2 자동으로 수집되는 정보

  • 기기 정보: 앱 버전, 안드로이드 버전 (앱 안정성 개선 목적)
  • 사용 통계: 앱 실행 횟수, 기능 사용 빈도 (앱 개선 목적)

2.3 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 연락처
  • 사진 및 미디어 파일

3. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

  • 생리 주기 추적 및 예측: 다음 생리 예정일 계산
  • 가임기 계산: 배란일 및 가임기 예측
  • 건강 인사이트 제공: 주기 패턴 분석 및 건강 정보 제공
  • 파트너 공유 기능: 사용자가 선택한 경우에만 데이터 공유
  • 앱 기능 개선: 익명화된 사용 통계를 통한 서비스 개선

4. 정보 저장 및 보안

4.1 로컬 저장

  • 모든 데이터는 사용자의 기기에만 저장됩니다
  • 클라우드 서버에 자동 업로드되지 않습니다
  • 앱을 삭제하면 모든 데이터가 완전히 삭제됩니다

4.2 암호화

  • 모든 민감한 데이터는 AES-256 암호화로 보호됩니다
  • 데이터베이스는 암호화되어 저장됩니다
  • 백업 파일도 암호화됩니다

4.3 백업

  • 사용자가 선택한 경우에만 백업 생성
  • 백업 파일은 사용자의 기기 또는 선택한 위치에 저장
  • 백업 파일은 암호화되어 저장됩니다

5. 정보 공유

5.1 제3자 공유 원칙

달의 기록은 사용자 데이터를 제3자에게 판매, 임대, 공유하지 않습니다.

5.2 파트너 공유 기능

사용자가 명시적으로 선택한 경우에만 데이터가 공유됩니다:

  • 공유되는 데이터: 최근 3개월 생리 기록 (시작일, 종료일), 다음 생리 예정일, 가임기 정보
  • 공유되지 않는 데이터: 증상, 기분, 메모 등 민감한 정보, 관계 기록의 상세 내용

5.3 법적 요구사항

다음의 경우에만 정보를 공개할 수 있습니다:

  • 법원 명령 또는 법적 절차에 따른 요구
  • 사용자 또는 제3자의 생명, 신체 보호를 위해 필요한 경우
  • 범죄 수사 목적으로 법 집행기관의 요청이 있는 경우

6. 사용자의 권리

6.1 데이터 접근 및 수정

언제든지 앱 내에서 본인의 데이터를 조회하고 수정할 수 있습니다.

6.2 데이터 삭제

  • 앱 내 "설정 → 데이터 삭제" 메뉴에서 모든 데이터 삭제 가능
  • 개별 기록을 선택적으로 삭제 가능
  • 앱을 삭제하면 모든 데이터가 완전히 제거됩니다

데이터 삭제 가이드 보기 →

6.3 데이터 내보내기

  • 앱 내 "백업 및 복원" 메뉴에서 데이터 내보내기 가능
  • JSON 형식으로 모든 데이터 다운로드
  • 다른 앱으로 데이터 이전 가능

6.4 동의 철회

  • 파트너 공유 등 동의한 기능을 언제든지 비활성화 가능
  • 설정 메뉴에서 각 기능별 권한 관리

7. 어린이 개인정보 보호

달의 기록은 만 14세 미만 어린이를 대상으로 하지 않으며, 고의로 만 14세 미만 어린이의 개인정보를 수집하지 않습니다. 만 14세 미만 어린이가 개인정보를 제공한 사실을 발견한 경우, 즉시 해당 정보를 삭제합니다.

8. 국제 사용자

달의 기록은 대한민국 개인정보 보호법을 준수합니다. 유럽연합(EU) 사용자의 경우 GDPR(일반 데이터 보호 규정)을 준수합니다.

EU 사용자의 추가 권리:
  • 데이터 이동권 (Data portability)
  • 잊힐 권리 (Right to be forgotten)
  • 처리 제한권 (Right to restriction of processing)

9. 개인정보처리방침 변경

본 개인정보처리방침은 관련 법령 또는 서비스 변경에 따라 수정될 수 있습니다.

  • 변경 시 앱 내 공지 또는 푸시 알림으로 안내
  • 중요한 변경 사항은 최소 7일 전 사전 공지
  • 변경된 방침은 공지 후 시행
